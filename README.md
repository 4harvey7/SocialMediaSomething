# SocialMediaSomething
📱 A .NET MAUI social feed app with post creation, image support, modal details view, and in-app commenting system.

📌 Social Feed App (MAUI)

A simple .NET MAUI social media feed application that allows users to create posts, view a feed, and interact with content through a modal-based interface.

The app features a modern social feed layout where users can add posts with a title, description, image, and visibility settings (Public/Private). Each post can be tapped to open a detailed view where users can read full content and add comments.

🚀 Features
📱 Social feed using CollectionView
➕ Add post via modal page
🖼️ Image support for posts
👁️ Public / Private visibility toggle
💬 Comment system inside post details
🕒 Auto-generated date and time
📄 Post detail modal view
🎯 Tap-to-view interaction
🛠️ Tech Stack
.NET MAUI
C#
XAML
📌 Purpose

This project demonstrates mobile UI development using .NET MAUI, focusing on navigation, data binding, modal interaction, and dynamic collection updates.
